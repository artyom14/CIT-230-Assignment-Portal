PARC:proximity

BYU-Idaho

This webpage exemplifies the proximity design principle because items connecting to each other are grouped close together. In the example above your eyes stop just two times. This items are in close proximity to each other, they looks like single visual component rather than several isolated components. It helps shape data, reduces clutter, and gives the viewer a clear construction.

PARC:Alignment

LDS

This webpage exemplifies the alignment design principle because nothing is placed on the page at will. We can see an example of aligning Images using the float property on an image tag . The placement of image on the page is defined to the right side and the text is wrapping around it. Elements of this page have visual connection with another element on the page. This creates a clean, sophisticated, fresh look.

PARC: Repetition

Feed music

This webpage exemplifies the repetition design principle because the same font styles for headers. For less important header, the font is made more transparent and smaller in a systematic way, but retain the other font properties. Also the background has repetitive graphic concepts. This develops the organization and strengthens the unity.
